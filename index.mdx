## What is the Mintie Calendar?

Mintie Calendar is a modern full-stack application designed to simplify task management and calendar event scheduling. Built with FastAPI and vanilla JavaScript, it offers both a user-friendly web interface and a comprehensive REST API for seamless integration.

## Technology Stack

Mintie Calendar is built with modern, reliable technologies:

- **Backend**: FastAPI (Python) with SQLAlchemy ORM
- **Database**: SQLite for simplicity and portability
- **Frontend**: Vanilla JavaScript with modern ES6+ features
- **API Documentation**: Auto-generated OpenAPI 3.1 specifications
